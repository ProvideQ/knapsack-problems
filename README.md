# knapsack-problems
Example Knapsack problems in the format used by our toolbox solvers.

That format is as follows:

```
{number of items}
{first item id} {first item value} {first item weight}
...
{last item id} {last item value} {last item weight}
{knapsack capacity}
```
example:
```
3
1 2 2
2 4 5
3 3 1
6
```
